3. Adventure
★ Adventure Class
1) wildPokemon : 야생포켓몬 이름 배열
   wildPokemonSkill : 야생포켓몬 스킬 배열
   RanNum : 랜덤 숫자
   RanNum set

2) [void] WildPoAppear : 야생포켓몬 출연 스크립트 및 선택지 제시
     야생포켓몬 나타났다! 스크립트
     선택지( 싸운다 / 도망친다 / 포켓몬 도감)
     선택지에 따른 if문(fight | Run )

3) [void] fight
   RanSkill : 범위내 랜덤 숫자로 인한 야생포켓몬 스킬 셋팅
     wildPower : 범위내 랜덤 숫자로 인한 야생포켓몬 힘 셋팅
     if문
     힘 비교 => 승리 => 경험치 상승  =>[일정경험치 도달시 Evolution_level2, Evolution_level3 실행]
                               =>[변화 없음]

          => 패배[변화없음]

4) [void] Run
   wildSpeed : 범위내 랜덤 숫자로 인한 야생포켓몬 속도 셋팅
    if문
    속도 비교 => 성공
                 => 실패 => fight

5) Evolution_level2 : 진화 ver1

6) Evolution_level3 : 진화 ver2
